### âœ… Complex Queries

#### 1. Nested Query (Find customers who made returns)
```sql
SELECT c.customer_id, c.name, c.phone_number
FROM Customer c
WHERE c.customer_id IN (
    SELECT DISTINCT s.customer_id 
    FROM Sale s
    WHERE s.sale_id IN (
        SELECT sale_id FROM Sale_Return
    )
);
```

#### 2. Join Query (Sales report with details)
```sql
SELECT 
    s.sale_id,
    c.name as customer_name,
    e.name as employee_name,
    p.name as product_name,
    si.quantity,
    si.price_at_time_of_sale,
    s.total_amount
FROM Sale s
LEFT JOIN Customer c ON s.customer_id = c.customer_id
LEFT JOIN Employee e ON s.employee_id = e.employee_id
JOIN Sale_Items si ON s.sale_id = si.sale_id
JOIN Product p ON si.product_id = p.product_id
ORDER BY s.sale_date DESC;
```

#### 3. Aggregate Query (Revenue by category)
```sql
SELECT 
    p.category,
    COUNT(DISTINCT si.sale_id) as total_sales,
    SUM(si.quantity) as items_sold,
    SUM(si.quantity * si.price_at_time_of_sale) as total_revenue,
    AVG(si.price_at_time_of_sale) as avg_price
FROM Sale_Items si
JOIN Product p ON si.product_id = p.product_id
GROUP BY p.category
ORDER BY total_revenue DESC;
```
